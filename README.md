# front-end

## clone

```
git clone --recursive https://github.com/VisualDataStruct/front-end.git
```

## update

```
git pull
git submodule update
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
